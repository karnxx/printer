# Conveytor – Conveyor 3D Printer

<img width="1696" height="1004" alt="image" src="https://github.com/user-attachments/assets/bcb73c6b-38bf-46c5-bcfc-140b19b5699a" />

Conveytor is an open-source conveyor-style 3D printer with a 300×300×250 build area and infinite forward printing.  
It uses a 45° belt setup, an E3D V6 hotend, a 5015 blower, a 3010 hotend fan, and runs Klipper on an SKR Mini E3 V3.

## Specs
- Print area: 300 × 300 × 250  
- Conveyor: unlimited length  
- Hotend: E3D V6  
- Cooling: 5015 part cooling, 3010 hotend fan  
- Firmware: Klipper  
- Board: SKR Mini E3 V3

## Overview
The printer uses standard belt-printer kinematics at 45°, allowing continuous prints or automatic part ejection.  
The design focuses on accessible parts, simple construction, and easy tuning under Klipper.
